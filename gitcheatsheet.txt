# git cheat sheet
# First add files you want to commit to repository:
#   git add .                   - For all files in your folder to repository
#   git add helloWorld.py       - For all specific file

# You can check what changes are staged for commit with:
#   git status

# Second make your commit for all files added, with -m "" add comment to your commit
#   git commit -m "Make a coment"

# Lastly upload your commit to github repository
#   git push origin main
